Learning HTML

Learning HTML doesn't take long at all, and it’s one of the easier markup languages! Be sure to be looking along at the code for this webpage to better understand HTML and CSS.

Basics of HTML are super easy to learn for everyone!

HTML websites are made up of elements. These elements put content into your website, content such as images to text to videos.

Elements

Most elements have an opening/closing tag. The opening tag looks like this <(element name)> and the closing tag looks like this .

Attributes

Attributes go inside elements to specify the element or add features to the tag. For example, you can set a link to an imagine using the href tag. Attributes usually look like <(attribute name)="">.

Preformatted Elements

Some HTML elements that are already built-in are headers (h1, h2, h3, etc.), paragraphs (p), page head (head), the tab title (title), and many more.

Divs

In addition to the preformatted ones, you can also split your content up using dividers, denoted by div. These just section off the website.

Classes & IDs

When using divs, you can use classes and IDs to specify the section of the webpage and provide details. These will also help for CSS!!

We can change colors with CSS!
The CSS content will be stored in a style tag.
Inside the style tag, you can code the CSS by adding the element name and then opening curly braces!
In these curly braces you will place the attributes and what they will be equal to.
Before we open the curly braces, you'll either put the element name, or a period and the class name, or a hashtag and the ID name.

We can also change the font style with CSS!
Use font-families and font-styles to change what they look like.
You can use the default fonts, or you can import fonts from Google Fonts!
